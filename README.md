# FrogGame - C# 🐸

A simple reaction-based game where the player must click a frog that randomly changes position to score points.

## 🎮 Features
- Easy & Hard modes
- Timer options (3 modes)
- High score tracking (Easy / Hard)
- Random position logic
- Windows desktop application (C#)

## 🧠 How to Play
Click the frog as fast as possible before time runs out.

## 🛠️ Technologies
- C#
- Windows Forms
